# Databites: The Epistemic Map of a Modern Data Engineer

Welcome to **Databites**, an open-source, public knowledge repository dedicated to mastering Power BI, Microsoft Fabric, [[DAX]], and core foundational Data Engineering principles.

This repository serves as a **central brain** and an **epistemic map**—detailing the specific scope, technical depth, and architectural paradigms required to excel as a Senior Enterprise Power BI / Fabric Developer. 

Whether you are a newcomer trying to understand how data flows from a semantic model to a visual, or an architect designing [[Composite Models|composite models]] and [[Incremental Refresh|incremental refresh]] policies, this repository is freely available for you to use, clone, and learn from.

## Getting Started

To systematically explore the knowledge base, start with the **[Master Index](_INDEX.md)**. 

To forcefully commit these complex architectural concepts to memory, you can import our **[Spaced Repetition (Anki)](Spaced%20Repetition%20(Anki).md)** flashcards directly into your learning app.

## The Scope (What's Covered)

- **Core Strategy & Preparation**: Mental models and meta-learning strategies.
- **Architecture & Data Modeling**: [[Kimball]] methodologies, Star Schemas, and [[The Anatomy of Keys|the anatomy of keys]].
- **The Modern Engineering Stack**: CI/CD, PBIP, [[git|Git]], [[TMDL]], and Microsoft [[Fabric Artifacts|Fabric artifacts]].
- **Power Query & [[SQL]]**: Maximizing [[Query Folding]], advanced M scripting, and API automation.
- **DAX & The Semantic Engine**: [[Vertipaq|VertiPaq]] internals, [[[[Context]] Transition]], [[Filter Propagation]], and [[Advanced [[DAX Patterns]]]].
- **Visualization & User Experience**: [[IBCS]], layout grids, pre-attentive attributes, and pixel-perfect [[Paginated Reports]].
- **Governance & Soft Skills**: [[Row Level Security]] (RLS), [[Requirements Engineering|requirements engineering]], and stakeholder management.

## How to Use This Repository

1. **Clone the repository** to your local machine.
2. **Open the folder in [Obsidian](https://obsidian.md/)** to instantly visualize the heavily interconnected knowledge web.
3. Use the graph view to explore how distinct concepts like "Filter Context" natively relate to "[[Calculation Groups]]" and "Row Level Security".
